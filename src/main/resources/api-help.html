🍽️ BudgetController REST API

Restoran POS sistemine uzaktan / mobil erişim için REST API. Birkaç açık endpoint (/api/ping, /api/login) dışında tüm istekler kimlik doğrulaması ister.

Kimlik Doğrulama Akışı (Bearer token)

Önerilen ve birincil akış:
  1. POST /api/login isteğiyle JSON body göndererek bir Bearer token alın.
  2. Sonraki tüm isteklerde header'a Authorization: Bearer <token> ekleyin.
  3. Çıkış için POST /api/logout ile token'ı iptal edin.
Token sunucu tarafında 8 saat absolute + 30 dakika idle TTL ile saklanır; süresi dolan token otomatik düşer. Token loglara yazılmaz.
Basic Auth (geriye uyumluluk): Eski istemciler hâlâ Authorization: Basic base64(username:password) göndererek kimlik doğrulayabilir; sunucu her istekte BCrypt karşılaştırması yapar (daha yavaş). Yeni istemciler Bearer token kullanmalıdır; Basic Auth desteği gelecekte kaldırılabilir.

Endpoint Listesi

MethodYolAçıklama
GET /api/pingHealth check (auth yok)
POST /api/login JSON body: {"username":"...","password":"..."} → cevapta token alanı döner
POST /api/logout Bearer token'ı iptal eder
GET /api/meGeçerli kullanıcı bilgisi
GET /api/productsTüm aktif ürünler
GET /api/tablesErişilebilir masalar
GET /api/tables/{tableNo}Masa detayı
POST /api/tables/{tableNo}/items Body: {"productId":1,"quantity":2}
POST /api/tables/{tableNo}/sale Body: {"method":"CASH|CREDIT_CARD|TRANSFER"}

Hızlı Test (curl)

# 1. Health check (auth yok)
curl http://localhost:7070/api/ping

# 2. Login → token al
TOKEN=$(curl -s -X POST http://localhost:7070/api/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"şifreniz"}' \
  | jq -r .token)

# 3. Sonraki istekler — Bearer token ile
curl -H "Authorization: Bearer $TOKEN" http://localhost:7070/api/me

# 4. Tüm masaları listele
curl -H "Authorization: Bearer $TOKEN" http://localhost:7070/api/tables

# 5. Sipariş ekle (garson token'ı ile)
curl -X POST http://localhost:7070/api/tables/101/items \
     -H "Authorization: Bearer $TOKEN" \
     -H "Content-Type: application/json" \
     -d '{"productId":1,"quantity":2}'

# 6. Çıkış
curl -X POST -H "Authorization: Bearer $TOKEN" http://localhost:7070/api/logout

# --- (Geriye uyumluluk için Basic Auth da çalışır) ---
curl -u admin:şifreniz http://localhost:7070/api/me
